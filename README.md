# Homemade-RAID-Array-LAMP-Wordpress
## Overview
Project documenting the setup of a RAID array and deployment of a WordPress site using a LAMP stack (Linux, Apache, MySQL, PHP). This project demonstrates storage redundancy, Linux server administration, and hosting a dynamic web application.
